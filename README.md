# ephjos.com

1. Setup ssh keys for target server
2. ssh into server
3. `mkdir -p /var/www/html`
3. Install blackholed
4. Start blackholed daemon
5. Run `bin/build && bin/deploy`, pointing `bin/deploy` at the target server
